Runge-Kutta Method for Systems of Differential Equations Solver

Runge-Kutta Method Calculations:

Step t ω₁ ω₂

Instructions:

  • Use `t` for the independent variable (time).
  • Use `y1`, `y2`, etc., for the dependent variables (the functions you're solving for).
  • Use `Math` for mathematical functions (e.g., `Math.exp`, `Math.log`, `Math.sin`).
  • Avoid using spaces between operators and numbers or variables.

Examples:

  • Linear System: \( y'_1 = t - y_1 \), \( y'_2 = y_1 - y_2 \) → { t - y1, y1 - y2 }
  • Exponential System: \( y'_1 = e^{t} - y_1 \), \( y'_2 = e^{t} - y_2 \) → { Math.exp(t) - y1, Math.exp(t) - y2 }
  • Logarithmic System: \( y'_1 = \ln(y_1) \), \( y'_2 = \ln(y_2) \) → { Math.log(y1), Math.log(y2) }
  • Trigonometric System: \( y'_1 = \sin(t) + \cos(y_1) \), \( y'_2 = \sin(t) + \cos(y_2) \) → { Math.sin(t) + Math.cos(y1), Math.sin(t) + Math.cos(y2) }
  • Hyperbolic System: \( y'_1 = \sinh(t) - \cosh(y_1) \), \( y'_2 = \sinh(t) - \cosh(y_2) \) → { Math.sinh(t) - Math.cosh(y1), Math.sinh(t) - Math.cosh(y2) }
  • Power System: \( y'_1 = t^2 + y_1^3 \), \( y'_2 = t^2 + y_2^3 \) → { Math.pow(t, 2) + Math.pow(y1, 3), Math.pow(t, 2) + Math.pow(y2, 3) }
  • Rational System: \( y'_1 = \frac{t}{y_1} \), \( y'_2 = \frac{t}{y_2} \) → { t / y1, t / y2 }
  • Piecewise System: \( y'_1 = \begin{cases} t & \text{if } t < 0 \\ y_1 & \text{otherwise} \end{cases}, \quad y'_2 = \begin{cases} t & \text{if } t < 0 \\ y_2 & \text{otherwise} \end{cases} \) → { (t < 0) ? t : y1, (t < 0) ? t : y2 }
